นโยบายความเป็นส่วนตัว

อัปเดตล่าสุด: 17 กรกฎาคม 2026

ViewTitle ("เรา") ให้บริการเปลี่ยนชื่อคลิป YouTube ของคุณให้แสดงยอดวิวล่าสุดโดยอัตโนมัติ เอกสารนี้อธิบายว่าเราเก็บและใช้ข้อมูลอะไรบ้าง

1. ข้อมูลที่เราเข้าถึงและเก็บ

  • อีเมล Google ของคุณ — เพื่อระบุตัวตนและติดต่อ
  • ข้อมูลคลิป YouTube — ยอดวิว, ชื่อคลิป, คำอธิบาย, แท็ก, หมวดหมู่ ของคลิปที่คุณเลือก
  • OAuth token — เพื่อให้ระบบเข้าถึง YouTube API แทนคุณได้ (เก็บไว้อย่างปลอดภัย ไม่เปิดเผย)
  • การตั้งค่าของคุณ — video ID, รูปแบบชื่อ, สถานะเปิด/ปิด

เรา ไม่เก็บรหัสผ่าน Google ของคุณ การล็อกอินทำผ่าน Google OAuth ทั้งหมด

2. เราใช้ข้อมูลทำอะไร

ใช้เพื่อให้บริการหลักเพียงอย่างเดียว คือ อ่านยอดวิวของคลิปที่คุณเลือก แล้วอัปเดตชื่อคลิปนั้น ตามรูปแบบที่คุณตั้งไว้ เป็นรอบทุก 30 นาที เราไม่ใช้ข้อมูลเพื่อการโฆษณาหรือวัตถุประสงค์อื่น

3. การใช้สิทธิ์ YouTube API

บริการนี้ใช้ YouTube API Services เมื่อใช้งานถือว่าคุณยอมรับ ข้อกำหนดของ YouTube และ นโยบายความเป็นส่วนตัวของ Google

Limited Use Disclosure: การใช้และถ่ายโอนข้อมูลที่ได้รับจาก Google APIs ของแอปนี้ เป็นไปตาม Google API Services User Data Policy รวมถึงข้อกำหนด Limited Use อย่างเคร่งครัด

4. การแบ่งปันข้อมูล

เราไม่ขาย ไม่แลกเปลี่ยน และไม่แบ่งปันข้อมูลของคุณกับบุคคลที่สาม ยกเว้นการเรียกใช้ Google/YouTube API ที่จำเป็นต่อการทำงานของบริการเท่านั้น

5. การเก็บและการลบข้อมูล

คุณสามารถลบบัญชีและข้อมูลทั้งหมดได้ตลอดเวลา ผ่านปุ่ม "ลบบัญชี" ในหน้าแดชบอร์ด ซึ่งจะลบข้อมูลของคุณออกจากระบบและถอนสิทธิ์ token ทันที หรือถอนสิทธิ์ได้เองที่ Google Account Permissions

6. ความปลอดภัย

เราจัดเก็บ token และข้อมูลบนเซิร์ฟเวอร์ที่มีการป้องกันตามสมควร การเชื่อมต่อทั้งหมดเข้ารหัสผ่าน HTTPS
